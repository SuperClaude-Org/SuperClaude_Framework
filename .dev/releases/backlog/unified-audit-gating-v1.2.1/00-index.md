# Audit Gate Backlog Package

This directory contains the design package for the Unified Audit Gating System.

## Contents
1. `01-requirements-spec.md`
   - Requirements-only spec with locked decisions and acceptance criteria.

2. `02-design-spec-v1.md`
   - Initial official design architecture.

3. `03-adversarial-review.md`
   - Structured strengths/weaknesses, risk register, recommendations, go/no-go.

4. `04-design-v1.1-delta-handoff.md`
   - Design delta + handoff packet for follow-on model iteration.

## Intent
Use this package as the canonical context for subsequent review, design refinement, and implementation planning for:
- `/sc:audit-gate` command
- supporting skill/protocol
- agent orchestration
- templates/refs
- sprint CLI integration
